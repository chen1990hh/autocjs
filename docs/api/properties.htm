Properties

defaults

Description

Type：
Object

静态属性，存储的是 AutocJS 对象默认配置选项。

Values

article：
页面中显示文章正文的 DOM 节点或者 ID 选择器。
selector：
article 节点下，要生成导航的标题标签的选择器。
title：
AutocJS 自动创建的导航菜单标题文字
isAnchorsOnly：
是否只创建标题链接
isAnimateScroll：
是否使用动画滚动定位
hasDirectoryInArticle：
是否在文章中创建目录导航
hasChapterCodeAtHeadings：
是否在文章标题中显示该标题的段落索引编号
hasChapterCodeInDirectory：
是否在导航菜单中显示段落索引编号
ANCHOR：
标题标签中创建的标题链接的 HTML 模板代码
WRAP：
AutocJS 菜单根节点的 HTML 模板代码
HEADER：
AutocJS 菜单标题栏的 HTML 模板代码
BODY：
AutocJS 菜单内容节点的 HTML 模板代码
FOOTER：
AutocJS 菜单页脚节点的 HTML 模板代码
SWITCHER：
AutocJS 菜单展开显示开关的 HTML 模板代码
TOP：
AutocJS 菜单返回顶部按钮的 HTML 模板代码
CHAPTERS：
AutocJS 导航目录列表的 HTML 模板代码
SUBJECTS：
AutocJS 导航子目录列表的 HTML 模板代码
CHAPTER：
AutocJS 导航段落章节的 HTML 模板代码
TEXT：
AutocJS 导航段落章节链接的 HTML 模板代码
CODE：
AutocJS 段落章节索引编码的 HTML 模板代码
OVERLAY：
AutocJS 菜单展开时遮罩层的 HTML 模板代码

version

Description

Type：
String

存储的是 AutocJS 当前的版本号。
