Properties

defaults

Description

Type：
Object

静态属性，存储的是 AutocJS 对象默认配置选项。

Values

article：
页面中显示文章正文的 DOM 节点或者 ID 选择器。
selector：
article 节点下，要生成导航的标题标签的选择器。
title：
AutocJS 自动创建的导航菜单标题文字
isAnchorsOnly：
是否只创建标题链接
isAnimateScroll：
是否使用动画滚动定位
hasDirectoryInArticle：
是否在文章中创建目录导航
hasCodeAtHeadings：
是否在文章标题中显示该标题的段落索引编号
ANCHOR：
标题标签中创建的标题链接的 HTML 模板代码
WRAP：
AutocJS 菜单根节点的 HTML 模板代码
HEADER：
AutocJS 菜单标题栏的 HTML 模板代码
BODY：
AutocJS 菜单内容节点的 HTML 模板代码
FOOTER：
AutocJS 菜单页脚节点的 HTML 模板代码
SWITCHER：
AutocJS 菜单展开显示开关的 HTML 模板代码
TOP：
AutocJS 菜单返回顶部按钮的 HTML 模板代码
CHAPTERS：
AutocJS 导航目录列表的 HTML 模板代码
SUBJECTS：
AutocJS 导航子目录列表的 HTML 模板代码
CHAPTER：
AutocJS 导航段落章节的 HTML 模板代码
TEXT：
AutocJS 导航段落章节链接的 HTML 模板代码
CODE：
AutocJS 段落章节索引编码的 HTML 模板代码
OVERLAY：
AutocJS 菜单展开时遮罩层的 HTML 模板代码

version

Description

Type：
String

存储的是 AutocJS 当前的版本号。

attributes

Description

Type：
Object

存储的是 AutocJS 对象当前的所有配置信息（具体信息请参考 defaults 属性）。

elements

Description

Type：
Array

存储的是 AutocJS 对象相关的所有 DOM 节点。

Values

article：
AutocJS 对象要收集的文章正文显示区域的 DOM 节点。
chapters：
AutocJS 对象将 hasDirectoryInArticle 设置为 true 时，在文章正文开始处创建的目录导航列表 DOM 节点。
wrap：
AutocJS 对象创建的目录导航菜单的根节点。
header：
AutocJS 对象创建的目录导航菜单的标题栏 DOM 节点。
body：
AutocJS 对象创建的目录导航菜单的正文内容 DOM 节点。
list：
AutocJS 对象创建的目录导航菜单的列表 DOM 节点。
footer：
AutocJS 对象创建的目录导航菜单的页脚 DOM 节点。
switcher：
AutocJS 对象创建的目录导航菜单的隐藏显示开关 DOM 节点。
top：
AutocJS 对象创建的目录导航菜单中的返回顶部 DOM 节点。
overlay：
AutocJS 对象创建的目录导航菜单的遮罩层 DOM 节点。

data

Description

Type：
String

存储的是 AutocJS 对象相关的所有数据信息。

Values

headings：
AutocJS 对象收集的所有文章标题节点。
chapters：
AutocJS 对象根据 data.headings 收集的（一一对应）的文章章节信息。

chapters 的数据示例如下：

{
   // id 编号
   id: 0,
   // 标题节点标签的层级级别
   level: 1,
   // 标题节点的文字
   text: '文章标题',
   // 标题节点 tagName.toUpperCase()
   tag: 'H1',
   // 标题节点的父级节点　id 编号
   pid: 0
}
anchors：
AutocJS 对象根据 data.headings 一一对应的创建的标题锚点链接 DOM 节点（NodeList）。
list：
AutocJS 对象根据 data.chapters 将它的值按照 pid 分组后的（二维数组）数据。
