Methods

init( options )

init( options ) 是 AutocJS 的初始化方法：

  1. 初始化配置参数；
  2. 初始化所有 DOM 元素；
  3. 初始化数据
  4. 绘制 UI 界面；
  5. 给 AutocJS 相关的 DOM 元素绑定事件处理器；

Parameters

{Object}
options - （可选）初始化的配置参数，详见 Configuration Options。

Returns

{AutocJS}
AutocJS 对象。

Code

var ArticleOutline = new AutocJS();

ArticleOutline.init({
    article: '#article',
    title: 'AutocJS v0.2.0'
});

reload( options )

reload( options ) 是（根据新的配置信息）重启程序方法：

  1. 先移除所有绘制的 DOM 元素和绑定的事件处理器；
  2. 重新初始化程序；

Parameters

{Object}
options - （可选）初始化的配置参数，详见 Configuration Options。

Returns

{AutocJS}
AutocJS 对象。

Code

var ArticleTOC = new AutocJS({
    isAnchorsOnly: false
});

ArticleTOC.reload({
    title: 'Reload AutocJS',
    isAnchorsOnly: true
});

get( prop )

get( prop ) 方法返回某个 attributes 属性。

Parameters

{String}
prop - 想要获取的 attributes 属性名称。

Returns

{String|Boolean}
attributes 属性值。

Code

var ArticleTOC = new AutocJS({
     article: '#article',
     title: 'AutocJS v0.2.0'
});

ArticleTOC.get('title'); // -> 'AutocJS v0.2.0'

chapters( headings )

chapters( headings ) 方法传入 headings 参数，则设置 data 属性，没有传入 headings 数据，则返回 headings() 方法收集的的标题节点分析整理的 chapters 数据。

Parameters

{Array}
headings - （可选）标题 DOM 元素集合。

Returns

{AutocJS|Array}
没有 headings　参数，返回　AutocJS　对象，否则返回　chapters 数据。

Code

var ArticleTOC = new AutocJS({
     article: '#article',
     title: 'AutocJS v0.2.0'
});

ArticleTOC.chapters(); // -> 返回　chapters 数据
ArticleTOC.chapters([{
    id: 0,
    level: 1,
    pid : -1,
    tag: "H1",
    text : "Methods",
    value : "anchor-0",
},
...,
{
    id: n,
    level: 4,
    pid : 0,
    tag: "H4",
    text : "A title",
    value : "anchor-n",
}]); // -> AutocJS
