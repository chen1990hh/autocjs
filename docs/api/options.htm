Options

article

Description

Type：
String | HTMLElement
Default：
''

必选，用来指定页面中显示文章正文的 DOM 节点或者 ID 选择器。如果没有指定它，则程序将不会执行。

Usage

As a id selector

new AutocJS({
    article: '#article'
    ...
});

As a DOM element

new AutocJS({
    article: $('#article')
    ...
});

selector

Description

Type：
String
Default：
'h1,h2,h3,h4,h5,h6'

可选，用来指定 article 节点下，要生成导航的标题标签的选择器。

Usage

Default selector

new AutocJS({
    article: '#article'
    // 不配置 selector 属性，即使用默认选择器
});

Customize selector

new AutocJS({
    article: $('#article'),
    selector: 'h3'
    ...
});

演示：Customize selector

title

Description

Type：
String
Default：
'Table of Contents'

可选，用来指定 AutocJS 自动创建的文章导读索引导航菜单的标题文字。

Usage

Default title

new AutocJS({
    article: '#article',
    // 不配置 title 属性，即使用默认标题文字
});

Customize title

new AutocJS({
    article: $('#article'),
    title: 'Customize title'
    ...
});

演示：Customize title

isAnchorsOnly

Description

Type：
Boolean
Default：
false

可选，用来指定是否只创建标题链接。

Usage

Create directory navigation and anchors

new AutocJS({
    article: '#article'
    // 不配置 isAnchorsOnly 属性，即会同时生成导航菜单和创建标题锚点链接
});

Create anchors only

new AutocJS({
    article: $('#article'),
    isOnlyAnchors: false
    ...
});

演示：Create anchors only

isAnimateScroll

Description

Type：
Boolean
Default：
true

可选，用来指定在点击段落索引导航链接时，是使用动画滚动定位，还是使用默认的锚点链接行为。

Usage

Scroll with animate effect

new AutocJS({
    article: '#article'
    // 不配置 isAnimateScroll 属性，即使用动画滚动定位
});

Act with anchor default behavior

new AutocJS({
    article: $('#article'),
    isAnimateScroll: false
    ...
});

演示：Positioning behavior

hasDirectoryInArticle

Description

Type：
Boolean
Default：
false

可选，用来指定是否在文章（开始位置）中创建目录导航。

Usage

No directory navigation in the article

new AutocJS({
    article: '#article'
    // 不配置 hasDirectoryInArticle 属性，就不会创建索引导航
});

has directory navigation in the article

new AutocJS({
    article: $('#article'),
    hasDirectoryInArticle: true
    ...
});

演示：Has directory navigation in the article

hasChapterCodeAtHeadings

Description

Type：
Boolean
Default：
false

可选，用来指定是否在文章标题中显示该标题的段落索引编号。

Usage

No index code at the headings

new AutocJS({
    article: '#article'
    // 不配置 hasTocInArticle 属性，就不会创建索引导航
});

Has index code at the headings

new AutocJS({
    article: $('#article'),
    hasCodeAtHeadings: true
    ...
});

演示：Has chapter code at the headings

ANCHOR

Description

Type：
String
Default：
'<a class="autocjs-anchor" aria-hidden="true"></a>'

可选，用来指定标题标签中创建的标题链接的 HTML 模板代码。

WRAP

Description

Type：
String
Default：
'<div id="toc" class="autocjs autocjs-hide" aria-hidden="true"></div>'

可选，用来指定 AutocJS 菜单根节点的 HTML 模板代码。

HEADER

Description

Type：
Boolean
Default：
'<h3 class="autocjs-hd" aria-hidden="true">{title}</h3>'

可选，用来指定 AutocJS 菜单内容节点的 HTML 模板代码。

BODY

Description

Type：
Boolean
Default：
'<nav class="autocjs-bd" aria-hidden="true"></nav>'

可选，用来指定 AutocJS 菜单内容节点的 HTML 模板代码。

FOOTER

Description

Type：
Boolean
Default：
'<div class="autocjs-ft" aria-hidden="true"></div>'

可选，用来指定 AutocJS 菜单页脚节点的 HTML 模板代码。

SWITCHER

Description

Type：
Boolean
Default：
'<h2 class="autocjs-switcher" title="Toggle Menu" aria-hidden="true">Ξ</h2>'

可选，用来指定 AutocJS 菜单展开显示开关的 HTML 模板代码。

TOP

Description

可选，用来指定 AutocJS 菜单返回顶部按钮的 HTML 模板代码。

Type：
Boolean
Default：
'<a class="autocjs-top" href="#top" aria-hidden="true">TOP</a>'

CHAPTERS

Description

Type：
Boolean
Default：
'<ol class="autocjs-chapters" aria-hidden="true"></ol>'

可选，用来指定 AutocJS 导航目录列表的 HTML 模板代码。

SUBJECTS

Description

Type：
Boolean
Default：
'<ol class="autocjs-subjects" aria-hidden="true"></ol>'

可选，用来指定 AutocJS 导航子目录列表的 HTML 模板代码。

CHAPTER

Description

Type：
Boolean
Default：
'<li class="autocjs-chapter" aria-hidden="true"></li>'

可选，用来指定 AutocJS 导航段落章节的 HTML 模板代码。

TEXT

Description

Type：
Boolean
Default：
'<a class="autocjs-text" aria-hidden="true"></a>'

可选，用来指定 AutocJS 导航段落章节链接的 HTML 模板代码。

CODE

Description

Type：
Boolean
Default：
'<em class="autocjs-code" aria-hidden="true"></em>'

可选，用来指定 AutocJS 段落章节索引编码的 HTML 模板代码。

OVERLAY

Description

Type：
Boolean
Default：
'<div class="autocjs-overlay autocjs-hide" aria-hidden="true"></div>'

可选，用来指定 AutocJS 菜单展开时遮罩层的 HTML 模板代码。
