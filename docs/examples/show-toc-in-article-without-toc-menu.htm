Examples

设置导航菜单的标题文本

AutocJS 可以使用 title 配置项设置导航菜单的标题文本，默认值：'Table of Contents'。

默认标题

new AutocJS({
    article: '#container'
});

演示地址：默认标题

自定义标题

new AutocJS({
    article: '#container',
    title: 'Examples'
});

演示地址：自定义标题

设置滚动模式

AutocJS 可以使用 isAnimateScroll 配置项设置段落定位滚动模式，true - 动画滚动定位 （默认值），false - 链接锚点定位。

锚点链接

new AutocJS({
    article: '#container',
    isAnimateScroll: false
});

演示地址：锚点链接

设置是否在文章中显示段落索引导航

AutocJS 可以使用 showTocInArticle 配置项是否在文章中显示段落索引导航，true - 索引导航 ，false - 不显示索引导航（默认值）。

在文章中显示段落索引导航，同时显示导航菜单

new AutocJS({
    article: '#container',
    showTocInArticle: true
});

演示地址：在文章中显示段落索引导航，同时显示导航菜单

在文章中显示段落索引导航，但不显示导航菜单

new AutocJS({
    article: '#container',
    showTocInArticle: true,
    onlyAnchors: true
});

演示地址：在文章中显示段落索引导航，但不显示导航菜单

设置是否显示导航菜单

AutocJS 可以使用 onlyAnchors 配置项是否显示导航菜单，true - 不显示菜单，false - 显示菜单 （默认值）。

显示导航菜单

new AutocJS({
    article: '#container',
    onlyAnchors: false
});

演示地址：显示导航菜单

只生成标题锚点

new AutocJS({
    article: '#container',
    onlyAnchors: true
});

演示地址：只生成标题锚点

设置是否在文章标题中显示段落索引信息

AutocJS 可以使用 showIndexAtAnchors 配置项是否在文章标题中显示段落索引信息，true - 显示，false - 不显示 （默认值）。

显示段落索引信息

new AutocJS({
    article: '#container',
    showIndexAtAnchors: true
});

演示地址：显示段落索引信息
